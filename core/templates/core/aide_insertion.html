Fonctions insertion

Planning, Émargement & Équipiers — Compagnons Bâtisseurs Bretagne

1. Accès et rôles

Le module Insertion est réservé aux utilisateurs ayant l'un des rôles suivants :

RôleAccès
AdminAccès complet — toutes les équipes
ResponsableAccès complet — toutes les équipes
RHAccès complet — toutes les équipes
EncadrantAccès à son équipe uniquement

La section Insertion apparaît dans le menu latéral uniquement pour ces rôles. Elle contient : Planning, Émargement, Feuilles de présence, Équipiers et un lien vers ce Manuel.

2. Planning

Vue d'ensemble

Le planning affiche un calendrier mensuel avec une ligne par équipe. Chaque barre colorée représente une affectation (un chantier planifié sur une période donnée).

  • Naviguez entre les mois avec les flèches ‹ et ›
  • Chaque barre indique le nom du chantier, le nombre de jours et le créneau de début (M = matin, A = après-midi)
  • Une icône de verrou sur une barre indique qu'au moins une présence a déjà été enregistrée en émargement — cette affectation ne peut plus être supprimée
Jours fériés et événements — les colonnes des jours non ouvrés (fériés, fermetures) apparaissent en gris clair et ne comptent pas dans le calcul des durées.

Affecter un chantier

Cliquez sur le bouton Affecter un chantier (ou directement sur le bouton + à droite d'une ligne d'équipe). Une fenêtre s'ouvre en 4 étapes :

Étape 1 — Choisir le chantier

Les cartes affichent les devis acceptés disponibles. Chaque carte indique :

  • La référence et le nom du client
  • Le chantier (intitulé du chantier)
  • Le montant MO total du devis et le MO restant à planifier
  • Un badge indiquant les équipes déjà assignées à ce devis

Filtrez les cartes en tapant dans le champ de recherche (client, chantier, référence). Cliquez sur l'icône ↗ pour ouvrir le devis dans un nouvel onglet.

Étape 2 — Choisir l'équipe

Chaque carte équipe affiche une estimation du nombre de jours nécessaires pour couvrir le MO restant, ainsi que le nombre de chantiers déjà en cours. Cliquez sur une équipe pour la sélectionner — vous passez automatiquement à l'étape suivante.

Étape 3 — Durée et tranche

Tranche : si le devis est divisé en phases (tranches), sélectionnez la tranche concernée, ou créez-en une nouvelle via + Nouvelle tranche. Si une seule tranche existe, elle est sélectionnée automatiquement.

Durée : cliquez sur Prendre tout pour utiliser le nombre de jours estimé (MO restant / taux jour / nb équipiers), ou saisissez un nombre de jours personnalisé.

Étape 4 — Date de démarrage

Le mini-calendrier affiche le mois en cours. Les cases colorées en jaune correspondent aux chantiers déjà planifiés pour cette équipe (survolez pour voir le libellé).

  • Cliquez sur un jour pour choisir la date de début
  • La date de fin est calculée automatiquement en jours ouvrés (hors fériés et événements)
  • Choisissez le créneau de démarrage (Matin ou Après-midi) — la fin est toujours en après-midi

Cliquez sur Enregistrer — la barre apparaît immédiatement sur le planning.

Déplacer / modifier une affectation

Glissez-déposez une barre pour la déplacer sur le planning :

  • Horizontalement — change la date de début (la durée est conservée)
  • Verticalement — change l'équipe assignée

En début de barre, un sélecteur de créneau (M / A) permet de modifier le créneau de démarrage.

Affectation verrouillée — une barre avec une icône de verrou ne peut pas être déplacée car des présences ont déjà été saisies en émargement.

Supprimer une affectation

Cliquez sur une barre pour ouvrir sa fiche, puis cliquez sur Supprimer. La suppression est impossible si des présences ont été enregistrées (barre verrouillée).

Événements d'équipe

Un événement représente une indisponibilité d'équipe (formation, fermeture, etc.). Cliquez sur le bouton + Événement (ou sur une cellule vide du calendrier) pour en créer un.

  • Les jours couverts par un événement sont grisés et ne comptent pas dans les durées des affectations
  • Sélectionnez une plage de jours par cliquer-glisser sur le calendrier pour pré-remplir les dates

Prêt d'équipier

Un prêt permet d'enregistrer qu'un équipier de votre équipe intervient temporairement dans une autre équipe (ou inversement).

  1. Cliquez sur le bouton Prêt équipier (ou glissez un équipier vers une autre ligne du planning)
  2. Sélectionnez l'équipier, l'équipe d'accueil et la période (cliquer-glisser sur le calendrier)
  3. Cliquez sur Enregistrer

L'équipier apparaît sur les deux lignes : dans son équipe d'origine (avec mention "en prêt") et dans l'équipe d'accueil.

Les jours de prêt sont pris en compte dans le calcul de la charge de l'équipe d'accueil et dans l'émargement des deux équipes concernées.

3. Émargement

Vue d'ensemble

L'émargement affiche une grille hebdomadaire par équipe. Chaque colonne correspond à une demi-journée (Matin / Après-midi). Les lignes sont les équipiers de l'équipe.

  • Naviguez entre les semaines avec les flèches ‹ et ›
  • Les cellules colorées en vert indiquent une présence enregistrée
  • Les cellules en gris clair correspondent aux demi-journées hors affectation planifiée
  • Le chantier afférent est affiché dans l'en-tête de colonne

Pointer une présence

Cliquez sur une cellule pour basculer la présence (présent / absent). Une cellule verte = présent, vide = absent.

Pour pointer plusieurs demi-journées d'un coup, maintenez le clic et glissez sur les cellules adjacentes.

Validation groupée — en bas de chaque colonne, un bouton permet de pointer / dépointer toute la colonne (toute l'équipe sur une demi-journée) en un clic.

Équipiers en prêt

Les équipiers en prêt depuis une autre équipe apparaissent dans la grille de l'équipe d'accueil avec un indicateur visuel. Leur présence se pointe de la même façon que les équipiers réguliers.

Dans l'équipe d'origine, les jours de prêt sont affichés mais grisés — la présence est gérée côté équipe d'accueil.

4. Équipiers

La page Équipiers liste les personnes en insertion rattachées à chaque équipe.

Ajouter un équipier

  1. Cliquez sur Nouvel équipier
  2. Renseignez le prénom, nom et l'équipe de rattachement
  3. Cliquez sur Enregistrer

Désactiver un équipier

Cliquez sur le bouton Actif / Inactif à droite de l'équipier. Un équipier inactif n'apparaît plus dans les grilles d'émargement ni dans les prêts, mais son historique de présences est conservé.

Nombre d'équipiers par équipe — le nombre affiché dans le planning (pour le calcul de la charge) correspond au champ nb_équipiers de l'équipe, défini dans l'administration. Il ne compte pas automatiquement les équipiers actifs de la liste.

5. Feuilles de présence

Vue d'ensemble

Les feuilles de présence mensuelles sont des documents réglementaires FSE/CISP. Elles sont générées automatiquement à partir des pointages saisis dans l'émargement hebdomadaire — aucune double saisie n'est nécessaire.

Accédez à la liste via Feuilles de présence dans le menu latéral. Sélectionnez une équipe et un mois pour voir le tableau de complétion de chaque équipier.

  • Badge ○ — aucune présence saisie ce mois
  • Badge ⏳ — pointage partiel (en cours)
  • Badge ✓ — toutes les demi-journées du mois pointées
Délai RH — les fiches doivent être remises avant le 27 du mois pour la paie du 1er jour ouvré du mois suivant.

Lire et compléter une fiche

Cliquez sur un équipier dans la liste pour ouvrir sa fiche individuelle. La fiche affiche une grille semaine par semaine (ISO) avec deux lignes par jour — Matin (M) et Après-midi (A).

  • Les cellules pré-remplies proviennent de l'émargement hebdomadaire
  • Les cellules à fond ocre appartiennent au mois précédent (semaine à cheval sur deux mois) — elles sont éditables mais ne modifient pas la fiche du mois précédent
  • Les cellules à fond gris sont hors-mois (non éditables)

Cliquez dans une cellule et saisissez un nombre d'heures ou un code d'absence. La sauvegarde est automatique (à la sortie de la cellule).

Valeurs non standard — une valeur différente du nombre d'heures par défaut de l'équipe s'affiche en gras italique pour attirer l'attention.

Codes de présence

Les codes suivants peuvent être saisis à la place d'un nombre d'heures :

CodeSignification
FJour férié légal (pré-rempli automatiquement)
RPont / Récupération (pré-rempli si un événement "Pont → Récup" est créé)
CCongé
MMaladie
ATAccident du travail
AAbsence non justifiée
AJAbsence justifiée
SFormation
PMSMPMise en situation en milieu professionnel
DEDépart anticipé
DIDisponibilité
Jours fériés — les jours fériés légaux (1er janvier, Pâques, 1er mai…) sont pré-remplis automatiquement avec le code F. Si votre équipe organise un pont, un événement de type "Pont → Récup" créé dans le Planning pré-remplira le code R. Ces pré-remplissages ne s'appliquent que si la cellule n'a pas encore de valeur saisie.

Imprimer

Cliquez sur le bouton Imprimer en haut de la fiche. La feuille est formatée en A4 paysage pour correspondre au document réglementaire : en-tête avec nom du chantier, ETI, mois, lignes de signature, légende des codes et logos des financeurs.

Pour imprimer toutes les fiches d'une équipe en une seule opération, utilisez le bouton Imprimer toutes les fiches sur la page de liste — chaque fiche occupe une page A4.
