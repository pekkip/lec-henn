Manuel utilisateur

Outil Devis & Facturation — Compagnons Bâtisseurs Bretagne

1. Connexion

L'outil est accessible depuis n'importe quel navigateur (Chrome, Firefox, Edge…) à l'adresse :

https://lec-henn-production.up.railway.app/login/

  1. Saisissez votre identifiant et votre mot de passe temporaire (reçus par email)
  2. Cliquez sur Se connecter
  3. Changez votre mot de passe dès la première connexion depuis votre profil (menu en haut à droite)
Mot de passe oublié ? Cliquez sur Mot de passe oublié ? sur la page de connexion et saisissez votre adresse @compagnonsbatisseurs.eu — un nouveau mot de passe temporaire vous sera envoyé par email.

2. Tableau de bord

Le tableau de bord s'affiche à la connexion. Il est composé de widgets (petites fenêtres) que vous choisissez et organisez vous-même : indicateurs chiffrés, listes, graphiques et activité récente.

Personnaliser votre tableau de bord

Cliquez sur le bouton Personnaliser (en haut à droite) pour passer en mode édition. Vous pouvez alors :

  • Réorganiser — glissez-déposez un widget pour le déplacer (poignée en haut à droite du widget) ;
  • Masquer — cliquez sur l'icône œil barré d'un widget pour le retirer (il repasse dans « Ajouter un widget ») ;
  • Ajouter — choisissez un widget dans le panneau Ajouter un widget en bas de page.

Cliquez sur Verrouiller pour enregistrer et quitter le mode édition. Votre disposition est mémorisée et vous est propre.

Portée des données (Tous / Mes données / Mon équipe)

Chaque widget possède un petit menu de portée (en haut à droite) :

  • Tous — toutes les données de l'association ;
  • Mes données — uniquement ce que vous avez créé ;
  • Mon équipe — votre équipe (vous, vos collègues d'équipe, vos techniciens).
Les widgets liés à la comptabilité (avoirs…) n'apparaissent que pour les comptables et administrateurs.

La barre du haut (topbar)

Toujours visible, elle contient le logo (cliquable pour revenir au tableau de bord), le titre de la page en cours, votre identifiant et le bouton de déconnexion.

Le menu latéral (sidebar)

  • Tableau de bord — vue d'ensemble
  • Devis — liste de tous les devis
  • Factures — factures liées à un devis (chantiers)
  • Avoirs — tous les avoirs (chantiers + compta)
  • Clients — liste des clients
  • Bibliothèque — votre catalogue personnel d'ouvrages
  • Aides travaux — bibliothèque des aides et subventions partagée

Les comptables et administrateurs disposent en plus d'une section Outils compta (factures structure, appels de convention) — voir le chapitre 6.

Le menu peut être replié (icônes seules) via le bouton Replier tout en bas du menu — pratique pour gagner de la place. L'état est mémorisé.

3. Les clients

Consulter et filtrer la liste

Cliquez sur Clients dans le menu latéral. Vous pouvez filtrer par :

  • Nom — recherche libre
  • Type — Particulier / Association / Bailleur / Collectivité / Autre
  • Code postal ou Département
  • Ville
  • Portée — Tous les clients / Mes clients / Mon équipe

Créer un client

Depuis la liste, cliquez sur Nouveau client. Vous pouvez aussi créer un client à la volée lors de la création d'un devis.

  • Saisissez le nom (obligatoire)
  • Renseignez le code postal — la ville est suggérée automatiquement
  • Ajoutez les informations de contact si disponibles
Éviter les doublons — avant de créer un client, tapez son nom dans le champ de recherche : l'autocomplétion affiche les clients existants.

4. Les devis

Liste et filtres

La liste affiche tous les devis de l'association, du plus récent au plus ancien. Filtres disponibles : territoire, service, équipe, statut, recherche libre (référence, client, chantier).

La colonne Reste à fact. indique le montant restant à facturer (orange = reste quelque chose, vert = tout est facturé).

Créer un devis

  1. Cliquez sur Nouveau devis
  2. Sélectionnez le client (tapez pour chercher, ou cliquez Parcourir pour filtrer, ou Nouveau client pour en créer un)
  3. Renseignez le chantier, l'équipe, le taux MO
  4. Cliquez sur Créer le devis

Vous êtes ensuite redirigé vers l'éditeur du devis.

L'éditeur de lignes

La page d'un devis est composée de plusieurs zones :

  • En-tête — client, chantier, équipe, taux MO, conditions, coordonnées CB
  • Barre d'outils — boutons pour ajouter des lignes
  • Éditeur de lignes — zone principale de construction du devis
  • Panneau bibliothèque — votre catalogue réutilisable (à gauche)
  • Récapitulatif — totaux MO, matériaux, financements, net client
  • Onglet Factures — factures liées à ce devis

Cliquez sur un champ (description, quantité, coût…) pour le modifier directement. Validez avec Entrée ou en cliquant ailleurs. Cliquez sur Sauvegarder pour enregistrer.

Réorganisez les lignes par glisser-déposer via l'icône ⠿ à gauche de chaque ligne.

Zones financement & subventions

Le devis dispose de deux zones de financement distinctes, activables indépendamment depuis la barre d'outils :

ZoneUsageIcône

Depuis chaque zone vous pouvez :

  • Glisser une aide depuis la section correspondante du panneau bibliothèque (Aides CBB ou Aides organismes)
  • Ajouter une ligne libre (Ajouter une ligne)
  • Supprimer la zone entière (Supprimer la zone)

Les montants des deux zones sont déduits dans le récapitulatif (Net client).

Vue client (PDF)

Cliquez sur le bouton Vue client pour afficher le document imprimable à envoyer au client. Utilisez la fonction d'impression du navigateur (Ctrl+P) pour générer un PDF.

Changez le statut depuis le sélecteur en haut de la page du devis.

Dupliquer un devis

Dans la liste des devis, cliquez sur l'icône de copie (deux feuilles) à droite du devis. Un nouveau devis brouillon est créé avec toutes les lignes du devis original.

5. Les factures

Types de documents

TypeUsage
AcompteVersement partiel avant démarrage des travaux
FactureFacture classique (intermédiaire ou solde)
AvoirAnnulation ou remboursement partiel

Créer une facture depuis un devis

  1. Ouvrez le devis concerné
  2. Cliquez sur l'onglet Factures
  3. Cliquez sur Nouvelle facture
  4. Choisissez le type (Acompte, Facture, Avoir) et renseignez le montant
  5. Cliquez sur Créer

La facture est créée en brouillon. Vous pouvez modifier son libellé, ses lignes et ses conditions avant de la valider.

Cycle de validation

StatutQui peut agirAction suivante
BrouillonÉquipe du devisValider (comptable) ou Bypass
ValidéeÉquipe du devisMarquer comme Envoyée
EnvoyéeÉquipe du devisMarquer comme Payée
Payée—État final

Le passage de Brouillon → Validée est réservé au comptable (ou un administrateur). C'est une étape de contrôle avant envoi au client.

Bypass de validation comptable

Si le comptable n'est pas disponible, un responsable ou administrateur peut contourner la validation via le bouton Bypass. Un code à 6 chiffres est envoyé par email au demandeur — il doit le saisir pour confirmer. L'action est tracée dans le journal d'audit.

Important — Le bypass est une exception, à utiliser avec parcimonie. Toutes les utilisations sont enregistrées et visibles dans le journal du devis.

Créer un avoir

Un avoir annule tout ou partie d'une facture (facture négative). Il se crée depuis une facture validée :

  1. Dans la liste des factures, repérez la facture concernée (statut Validée, Envoyée ou Payée)
  2. Cliquez sur l'icône ↩ Créer un avoir
  3. L'avoir reprend automatiquement les lignes de la facture avec les quantités inversées (ex. 20 m² → −20 m²)
  4. Ajustez les quantités si vous n'annulez qu'une partie (ex. corrigez à −10 m²), puis Sauvegarder et Valider

Une facture qui a donné lieu à un avoir affiche un badge rouge ↩ Avoir à côté de son statut (cliquable pour ouvrir l'avoir). Tous les avoirs sont regroupés dans la rubrique Avoirs du menu (chapitre commun aux factures de chantier et aux factures compta).

Numérotation — les avoirs ont leur propre série AV-2026-001, attribuée à la validation.

6. Outils compta (comptable / admin)

La section Outils compta du menu (visible uniquement pour les comptables et administrateurs) permet de facturer directement, sans passer par un devis. Deux outils sont disponibles : Factures structure et Appels de convention.

Ces documents s'adressent à des structures (collectivités, bailleurs, associations…) et reprennent la même présentation que les factures classiques. Ils n'ont pas d'adresse de chantier.

Factures structure

  1. Menu Outils compta → Factures Structure, puis Nouvelle facture
  2. Choisissez (ou créez) le client et, si besoin, un service / interlocuteur
  3. Renseignez l'objet et l'échéance, puis Créer
  4. Construisez les lignes dans l'éditeur, puis Sauvegarder et Valider

Les factures structure partagent la même numérotation que les factures classiques (FAC-2026-…, attribuée à la validation).

Appels de convention

Même fonctionnement que les factures structure, via Outils compta → Appels de convention. Seule différence : un préfixe de numérotation dédié (APP-2026-…) et le titre « Facture d'appel de convention » sur l'édition client.

L'éditeur (titres & forfaits)

L'éditeur des outils compta est volontairement simplifié — pas de distinction main-d'œuvre / matériaux. Deux types de lignes :

  • Titre — en-tête de section qui regroupe des forfaits
  • Forfait — une ligne avec description, quantité, unité et prix unitaire

Sur chaque ligne : glisser-déposer via la poignée ⠿ pour réorganiser, dupliquer (l'icône copie — un titre est dupliqué avec tous ses forfaits) et supprimer. Depuis la liste, l'icône copie duplique une facture entière en nouveau brouillon.

Proforma — tant qu'une facture est en brouillon, son édition client porte un numéro provisoire PF-… et la mention « proforma ». Le numéro définitif est attribué à la validation.

Outils de saisie rapide (Excel)

Deux boutons dans la barre de l'éditeur permettent d'importer des données depuis Excel en une seule opération :

BoutonUsage
Coller depuis Excel Coller n'importe quel tableau Excel → chaque ligne devient un forfait. Les dates (jj/mm/aaaa ou aaaa-mm-jj) sont détectées et normalisées. Les colonnes sont alignées en monospace.
Calcul Tutorat Outil dédié aux services civiques. Coller la liste des volontaires (colonnes : Secteur (optionnel), Nom, Prénom (ou Nom Prénom en 1 colonne), Date début, Date fin) puis choisir le trimestre, l'année et le taux mensuel. L'outil calcule le nombre de jours via la formule JOURS360 et génère automatiquement un titre Xe trimestre YYYY — SERVICES CIVIQUES avec une ligne de repère des colonnes et un forfait par personne (quantité = jours, unité J, prix unitaire = taux / 30).
Format de la colonne Secteur (Calcul Tutorat) — si la première colonne est un nombre (code secteur), elle est détectée automatiquement. Si les secteurs sont absents, collez directement Nom, Prénom (ou Nom+Prénom), Date début, Date fin.

Contacts / services d'un client

Une même structure (ex. une collectivité) peut avoir plusieurs services destinataires. Vous pouvez attacher à un client un carnet de contacts (service, interlocuteur, email, téléphone) :

  • À la création d'une facture compta — bouton Nouveau contact sous le sélecteur
  • Depuis la fiche client — la modale de modification (admin) liste et permet d'ajouter les contacts

C'est optionnel : un particulier ou une association n'a généralement aucun contact à part. Le champ Type du client (Particulier / Association / Bailleur / Collectivité / Autre) permet de filtrer la liste des clients.

Avoirs sur factures compta

Comme pour les factures de chantier, vous pouvez créer un avoir depuis une facture structure ou un appel de convention validé (voir Créer un avoir). L'avoir reprend les lignes en quantités inversées, et apparaît à la fois dans la liste Avoirs et dans la liste de l'outil concerné (avec une icône indiquant l'origine).

7. Ma bibliothèque personnelle

La bibliothèque est accessible depuis le menu latéral. C'est votre catalogue personnel d'ouvrages réutilisables d'un devis à l'autre.

Organiser sa bibliothèque

La bibliothèque fonctionne comme l'éditeur de devis : créez des titres pour regrouper vos ouvrages par catégories (ex : Maçonnerie, Plomberie…), ajoutez des ouvrages sous chaque titre, et réorganisez par glisser-déposer.

Utiliser un ouvrage dans un devis

Dans l'éditeur de devis, le panneau bibliothèque s'affiche à gauche. Glissez un ouvrage depuis ce panneau vers la zone du devis pour l'insérer. Le taux MO du devis est automatiquement appliqué.

Enregistrer un ouvrage depuis un devis

Dans l'éditeur, glissez une ligne depuis le devis vers le panneau bibliothèque à gauche. L'ouvrage est automatiquement enregistré dans votre bibliothèque.

Ajuster les tarifs

  • Taux MO — modifiez le taux en haut de la bibliothèque et appliquez-le à toutes les lignes MO en une action
  • % matériaux — dans chaque section, le champ % permet d'ajuster les prix matériaux de toute la catégorie

8. Bibliothèque Aides (partagée)

Accessible depuis 🎁 Aides dans le menu latéral. Contrairement à la bibliothèque personnelle, la bibliothèque Aides est partagée entre tous les utilisateurs.

Elle regroupe les aides financières et subventions connues (organisme, montant, type : MO / MAT / global / organisme). Tout le monde peut créer ou supprimer des aides.

Les aides sont réparties en deux catégories dans le panneau bibliothèque :

CatégorieTypesZone cible dans le devis
Aides CBB 🎁FIN, FMO, FMATZone Aide travaux CBB
Aides organismes 🏦FINXZone Financements organismes

Utiliser une aide dans un devis

  1. Activez la zone de financement correspondante dans l'éditeur du devis (Aide travaux CBB ou Financements organismes)
  2. La section Aides apparaît en bas du panneau bibliothèque avec les deux catégories
  3. Glissez une aide vers la zone correspondante, ou cliquez dessus pour l'insérer directement

Une ligne liée à une aide affiche un indicateur 🔗. Le montant reste modifiable sur le devis.

10. Gestion des utilisateurs (admin / responsable)

Accessible depuis Utilisateurs dans le menu latéral (visible uniquement pour les admins et responsables).

Rôles

RôleDroits
TechnicienCrée et modifie les devis de son équipe. Consulte tous les devis en lecture seule.
ResponsableIdem technicien + gère les utilisateurs de ses équipes + peut bypasser la validation comptable.
ComptableConsulte tous les devis (lecture seule) + valide les factures + accède aux Outils compta (factures structure, appels de convention, avoirs).
AdminAccès complet — tous les droits sur tous les objets.

Créer un utilisateur

  1. Cliquez sur Nouvel utilisateur
  2. Renseignez les informations (prénom, nom, email, identifiant, rôle, équipes)
  3. Cliquez sur Créer l'utilisateur

Si une adresse email est renseignée, un email d'invitation est envoyé automatiquement avec le mot de passe temporaire. Sinon, le mot de passe s'affiche une seule fois à l'écran.

Modifier ou désactiver un utilisateur

Dans la liste des utilisateurs, cliquez sur l'icône de modification pour changer le rôle, les équipes ou le responsable. Pour désactiver un compte, utilisez le bouton Désactiver — l'utilisateur ne peut plus se connecter mais ses données sont conservées.

Pas de suppression — les comptes sont désactivés, jamais supprimés, pour préserver l'historique des devis et factures.
