Outil Devis & Facturation — Compagnons Bâtisseurs Bretagne

1. Connexion

L'outil est accessible depuis n'importe quel navigateur (Chrome, Firefox, Edge…) à l'adresse :

{{ site_url }}/login/

  1. Saisissez votre identifiant et votre mot de passe temporaire (communiqués par votre administrateur) ;
  2. Cliquez sur Se connecter ;
  3. Changez votre mot de passe dès la première connexion depuis votre profil (menu en haut à droite).
Mot de passe oublié ? Pendant la phase de test, contactez votre administrateur : il vous régénère un mot de passe temporaire et vous le transmet directement.

2. Tableau de bord

Le tableau de bord s'affiche à la connexion. Il est composé de widgets (petites fenêtres) que vous choisissez et organisez vous-même : indicateurs chiffrés, listes, graphiques et activité récente.

Personnaliser votre tableau de bord

Cliquez sur le bouton Personnaliser (en haut à droite) pour passer en mode édition. Vous pouvez alors :

  • Réorganiser — glissez-déposez un widget pour le déplacer (poignée en haut à droite du widget) ;
  • Masquer — cliquez sur l'icône œil barré d'un widget pour le retirer (il repasse dans « Ajouter un widget ») ;
  • Ajouter — choisissez un widget dans le panneau Ajouter un widget en bas de page.

Cliquez sur Verrouiller pour enregistrer et quitter le mode édition. Votre disposition est mémorisée et vous est propre.

Portée des données (Tous / Mes données / Mon équipe)

Chaque widget possède un petit menu de portée (en haut à droite) :

  • Tous — toutes les données de l'association ;
  • Mes données — uniquement ce que vous avez créé ;
  • Mon équipe — votre équipe (vous, vos collègues d'équipe, vos techniciens).
Les widgets liés à la comptabilité n'apparaissent que pour les comptables et administrateurs. Des widgets de production (suivi des chantiers d'insertion) apparaissent pour les membres des services concernés.

La barre du haut (topbar)

Toujours visible, elle contient le logo (cliquable pour revenir au tableau de bord), le titre de la page en cours, votre identifiant (avec un liseré de votre couleur) et le bouton de déconnexion.

Le menu latéral (sidebar)

  • Tableau de bord — vue d'ensemble
  • Devis — liste de tous les devis
  • Factures — factures liées à un devis (chantiers)
  • Avoirs — tous les avoirs (chantiers + compta)
  • Clients — liste des clients
  • Bibliothèque — votre catalogue personnel d'ouvrages
  • Aides travaux — bibliothèque des aides et subventions partagée
  • Manuel — ce document

Les comptables et administrateurs disposent en plus d'une section Outils compta ; les rôles insertion voient une section Insertion (documentée dans le manuel dédié).

Le menu peut être replié (icônes seules) via le bouton Replier tout en bas — pratique pour gagner de la place. L'état est mémorisé.

3. Les clients

Consulter et filtrer la liste

Cliquez sur Clients dans le menu latéral. Vous pouvez filtrer par :

  • Nom — recherche libre
  • Type — Particulier / Association / Bailleur / Collectivité / Autre
  • Code postal ou Département
  • Ville
  • Portée — Tous les clients / Mes clients / Mon équipe

Créer un client

Depuis la liste, cliquez sur Nouveau client. Vous pouvez aussi créer un client à la volée lors de la création d'un devis (voir Créer un devis).

  • Saisissez le nom / organisation (obligatoire) ;
  • Renseignez le code postal — la ville est suggérée automatiquement ;
  • Ajoutez les informations de contact si disponibles.
Éviter les doublons — avant de créer un client, tapez son nom dans le champ de recherche : l'autocomplétion affiche les clients existants.

Tout utilisateur connecté peut créer et modifier un client. La suppression reste réservée aux administrateurs.

4. En bref : du devis à la facture

Le parcours type, du premier contact au règlement :

Parcours d'un chantier
Clientfiche client → Devisbrouillon → accepté → Facture(s)acompte, intermédiaire, solde → Avoirsi correction
Un même devis peut donner lieu à plusieurs factures (un acompte, des factures intermédiaires, une facture de solde). Chaque facture peut, si besoin, être corrigée par un avoir.

Les sections suivantes détaillent chaque étape : construire le devis, créer les factures, gérer la facturation partielle et les acomptes.

5. Les devis

Liste et filtres

La liste affiche tous les devis de l'association, du plus récent au plus ancien. Filtres disponibles : territoire, service, équipe, statut, auteur, recherche libre (référence, client, chantier).

La colonne Reste à fact. indique le montant restant à facturer (orange = il reste quelque chose, vert = tout est facturé).

Créer un devis — choisir le client

  1. Cliquez sur Nouveau devis ;
  2. Sélectionnez le client :
    • tapez son nom dans le champ de recherche (autocomplétion), ou
    • cliquez Parcourir pour ouvrir la liste filtrable, ou
    • cliquez Nouveau client pour le créer sans quitter la page (voir Créer un client) ;
  3. Renseignez le chantier / objet, l'équipe, la validité et le taux horaire MO ;
  4. Cliquez sur Créer le devis — vous arrivez dans l'éditeur.
Sélection obligatoire — le devis ne peut pas être créé sans un client choisi dans la liste : taper un nom ne suffit pas, il faut le sélectionner.

L'éditeur de lignes

La page d'un devis s'organise en onglets :

  • En-tête — client, chantier, équipe, taux MO, conditions, coordonnées CB ;
  • Édition — la zone principale de construction du devis (barre d'outils + arbre de lignes + panneau bibliothèque) ;
  • Factures — les factures liées à ce devis (voir chapitre 6) ;
  • Journal — l'historique des actions.

Dans l'onglet Édition, cliquez sur un champ (description, quantité, coût…) pour le modifier directement. Validez avec Entrée ou en cliquant ailleurs. Cliquez sur Enregistrer les lignes (ou Ctrl+S) ; une confirmation apparaît brièvement en bas à droite. Réorganisez les lignes par glisser-déposer via la poignée ⠿ à gauche de chaque ligne.

Aucune perte de saisie — si vous tentez de fermer l'onglet, de recharger ou de revenir en arrière alors que des modifications ne sont pas enregistrées, le navigateur demande confirmation. (Vaut aussi pour l'éditeur des factures et la bibliothèque.)

Types de lignes

Le devis est un arbre : des titres regroupent des ouvrages, et un ouvrage composite contient lui-même sa main-d'œuvre et ses matériaux. Voici la mini-maquette d'un devis et la signification de chaque type :

TypeUsage
TITRESection qui regroupe des lignes sous un intitulé. N'a pas de prix propre : il totalise ses lignes.
S Ouvrage simpleUn ouvrage à prix unitaire global (quantité × prix). Le plus rapide quand vous connaissez directement le prix au m², à l'unité…
C Ouvrage compositeUn ouvrage décomposé en main-d'œuvre + matériaux : son prix est calculé à partir des lignes MO et MAT qu'il contient.

Simple ou composite ? Utilisez un ouvrage simple quand vous avez un prix tout compris ; un composite quand vous voulez détailler la main-d'œuvre et les matériaux (et piloter la marge via le taux horaire).

Sous chaque titre, des boutons permettent d'ajouter un sous-titre, un forfait, un ouvrage simple, un ouvrage composite ou un saut de page.

Saut de page — insère une coupure qui force un changement de page dans le PDF client. Il n'a pas de prix et n'entre pas dans les totaux.

Utiliser la bibliothèque

Dans l'onglet Édition, un panneau bibliothèque est affiché sur le côté : c'est votre catalogue d'ouvrages réutilisables.

  • Insérer — glissez un ouvrage (ou un groupe) du panneau vers le devis ; le taux MO du devis est appliqué automatiquement ;
  • Enregistrer — glissez une ligne du devis vers le panneau pour l'ajouter à votre bibliothèque. Glissez un titre entier pour en faire un groupe réutilisable d'un bloc.

Détails et organisation : voir Ma bibliothèque.

Zones financement & subventions

Le devis dispose de deux zones de financement distinctes, activables indépendamment depuis la barre d'outils :

BoutonZoneUsage
🎁 Aide CBBAide travaux CBBAides attribuées par CBB (FIN / FMO / FMAT)
🏦 FinancementsFinancements organismesFinancements versés à CBB par des organismes tiers (collectivités, bailleurs…) — type FINX

Dans chaque zone vous pouvez glisser une aide depuis le panneau bibliothèque, ajouter une ligne libre, ou supprimer la zone entière. Les montants sont déduits dans le récapitulatif (Net client). Voir aussi Bibliothèque Aides.

Vue client (PDF)

Cliquez sur Vue client pour afficher le document à envoyer au client. Vous pouvez :

  • Télécharger le PDF (bouton dédié — document généré côté serveur, prêt à envoyer) ;
  • ou utiliser l'impression du navigateur (Ctrl+P).

Dupliquer un devis

Dans la liste des devis, cliquez sur l'icône de copie (deux feuilles) à droite du devis. Un nouveau devis brouillon est créé avec toutes les lignes du devis original.

Importer des devis existants

En haut de la liste des devis, deux boutons permettent d'importer des devis établis sur EBP :

Contrôle des totaux — si le total reconstitué ne correspond pas exactement à celui du document d'origine, l'écart est signalé par un préfixe ⚠ ÉCART (sur l'objet du devis ou sur la ligne concernée) afin que vous le vérifiiez.

Les devis importés conservent leur numéro d'origine. Pour un gros volume à importer, déposez les fichiers sur le SharePoint et transmettez le lien à l'administrateur (un outil en lot existe côté administration).

6. Les factures

Les factures de chantier se créent depuis un devis accepté, dans son onglet Factures. Un devis peut donner lieu à plusieurs factures successives jusqu'à couvrir son montant total.

Créer une facture depuis le devis

  1. Ouvrez le devis, onglet Factures (il rappelle Total brut, Facturé et Reste) ;
  2. Cliquez sur Nouvelle facture ;
  3. Choisissez le type :
    • Facture — les lignes sont pré-remplies depuis le devis (voir Facturation partielle) ; renseignez l'échéance ;
    • Acompte — saisissez un montant libre (voir Acomptes) ;
  4. Cliquez sur Créer en brouillon.

La facture est créée en brouillon. Ouvrez-la pour ajuster ses lignes, son libellé et ses conditions, puis suivez le cycle de validation.

Facturation partielle — lignes non facturées

Vous n'êtes pas obligé de tout facturer d'un coup. Dans l'éditeur d'une facture, chaque ligne a une quantité à facturer :

  • Mettez une ligne à 0 pour ne pas la facturer cette fois ; un titre entièrement à 0 se replie (rien n'apparaît sur le document) ;
  • La quantité restante (devis − déjà facturé) est reportée automatiquement sur la facture suivante ;
  • Chaque titre affiche son historique : « Facturé X € le … · réf », pour savoir ce qui a déjà été facturé et par quelle facture.
Schéma — un devis facturé en deux fois
Facture 1 (intermédiaire)
Maçonnerie — gros œuvre12 / 12 m²
Peinture — finitions0 / 40 m²
Facturé1 530,00 €
Facture 2 (solde)
Maçonnerie — gros œuvreFacturé le 12/03 · FA00042
Peinture — finitions40 / 40 m²
Facturé1 200,00 €
La peinture, non facturée sur la facture 1 (qté 0, grisée), revient automatiquement avec sa quantité restante sur la facture 2.

Acomptes

Un acompte est une avance versée par le client. Ce n'est pas une facturation en plus : la facture de solde porte le montant plein du devis, et l'acompte déjà versé en est déduit pour obtenir le « solde à régler ».

Schéma — acompte puis solde (devis 3 000 €)
Acompte 600 €avance versée + Facture de soldemontant plein 3 000 € →− acompte Solde à régler2 400 €
L'acompte (600 €) n'est pas ajouté aux 3 000 € : il est retranché du solde. Total encaissé = 600 + 2 400 = 3 000 €.
Verrou de cohérence — la validation d'une facture de solde est bloquée tant qu'un acompte émis n'est pas marqué payé. Pour débloquer : sur la ligne de l'acompte (statut « Envoyé »), cliquez sur l'icône ✓ pour le marquer payé, puis validez la facture de solde.

Cycle de validation

Schéma — statuts d'une facture
Brouillon →comptable Validée →envoi client Envoyée →règlement Payée
StatutQui peut agirAction suivante
BrouillonÉquipe du devisValider (comptable) ou Bypass
ValidéeÉquipe du devisMarquer comme Envoyée
EnvoyéeÉquipe du devisMarquer comme Payée
Payée—État final

Le passage Brouillon → Validée est réservé au comptable (ou un administrateur) : c'est l'étape de contrôle avant envoi. Tant qu'une facture est en brouillon, son aperçu porte un numéro provisoire PF-… et la mention « proforma » (le numéro définitif est attribué à la validation — voir Numérotation).

Bypass de validation comptable

Si le comptable n'est pas disponible, un responsable ou administrateur peut contourner la validation via le bouton Bypass. L'action est tracée dans le journal d'audit.

Phase de test — le bypass repose sur l'envoi d'un code par email, dont la délivrance vers les adresses internes n'est pas garantie actuellement. À utiliser avec parcimonie ; en cas de souci, rapprochez-vous de l'administrateur.

Créer un avoir

Un avoir annule tout ou partie d'une facture (facture négative). Il se crée depuis une facture validée :

  1. Repérez la facture concernée (statut Validée, Envoyée ou Payée) ;
  2. Cliquez sur l'icône ↩ Créer un avoir ;
  3. L'avoir reprend les lignes de la facture avec les quantités inversées (ex. 20 m² → −20 m²) ;
  4. Ajustez les quantités pour n'annuler qu'une partie si besoin, puis Enregistrer et Valider.

Une facture qui a donné lieu à un avoir affiche un badge rouge ↩ Avoir (cliquable). Tous les avoirs sont regroupés dans la rubrique Avoirs du menu.

7. Outils compta (comptable / admin)

La section Outils compta du menu (visible uniquement pour les comptables et administrateurs) permet de facturer directement, sans passer par un devis. Deux outils : Factures structure et Appels de convention. Ces documents s'adressent à des structures (collectivités, bailleurs, associations…) et n'ont pas d'adresse de chantier.

Factures structure

  1. Menu Outils compta → Factures Structure, puis Nouvelle facture ;
  2. Choisissez (ou créez) le client et, si besoin, un service / interlocuteur ;
  3. Renseignez l'objet et l'échéance, puis Créer ;
  4. Construisez les lignes dans l'éditeur, puis Enregistrer et Valider.

À la validation, elles reçoivent un numéro de la forme FACTURE-ST##### (voir Numérotation).

Appels de convention

Même fonctionnement, via Outils compta → Appels de convention. Le document client porte le titre « Facture d'appel de convention » et conserve une numérotation dédiée (APP-AAAA-NNN).

L'éditeur (titres & forfaits)

L'éditeur des outils compta est volontairement simplifié — pas de distinction main-d'œuvre / matériaux. Deux types de lignes :

  • Titre — en-tête de section qui regroupe des forfaits ;
  • Forfait — une ligne avec description, quantité, unité et prix unitaire.

Sur chaque ligne : glisser-déposer via la poignée ⠿, dupliquer (un titre est dupliqué avec ses forfaits) et supprimer. Depuis la liste, l'icône copie duplique une facture entière en nouveau brouillon.

Proforma — tant qu'une facture est en brouillon, son édition client porte un numéro provisoire PF-… et la mention « proforma ». Le numéro définitif est attribué à la validation.

Outils de saisie rapide (Excel)

Deux boutons dans la barre de l'éditeur permettent d'importer des données depuis Excel en une seule opération :

BoutonUsage
Coller depuis Excel Coller n'importe quel tableau Excel → chaque ligne devient un forfait. Les dates (jj/mm/aaaa ou aaaa-mm-jj) sont détectées et normalisées. Les colonnes sont alignées.
Calcul Tutorat Outil dédié aux services civiques. Coller la liste des volontaires (colonnes : Secteur (optionnel), Nom, Prénom — ou Nom Prénom en une colonne —, Date début, Date fin) puis choisir le trimestre, l'année et le taux mensuel. L'outil calcule le nombre de jours (méthode JOURS360) et génère un titre Xe trimestre AAAA — SERVICES CIVIQUES avec un forfait par personne.
Colonne Secteur (Calcul Tutorat) — si la première colonne est un nombre (code secteur), elle est détectée automatiquement. Sinon, collez directement Nom, Prénom, Date début, Date fin.

Contacts / services d'un client

Une même structure (ex. une collectivité) peut avoir plusieurs services destinataires. Vous pouvez attacher à un client un carnet de contacts (service, interlocuteur, email, téléphone) :

  • à la création d'une facture compta — bouton Nouveau contact sous le sélecteur ;
  • depuis la fiche client — la modale de modification liste et permet d'ajouter les contacts.

C'est optionnel : un particulier ou une association n'a généralement aucun contact à part. Le champ Type du client (Particulier / Association / Bailleur / Collectivité / Autre) permet de filtrer la liste.

Avoirs sur factures compta

Comme pour les factures de chantier, vous pouvez créer un avoir depuis une facture structure ou un appel de convention validé (voir Créer un avoir). L'avoir reprend les lignes en quantités inversées et apparaît à la fois dans la liste Avoirs et dans la liste de l'outil concerné.

8. Ma bibliothèque personnelle

La bibliothèque est accessible depuis le menu latéral. C'est votre catalogue personnel d'ouvrages réutilisables d'un devis à l'autre.

Organiser sa bibliothèque

La bibliothèque fonctionne comme l'éditeur de devis : créez des catégories pour regrouper vos ouvrages (Maçonnerie, Plomberie…), ajoutez des ouvrages sous chaque catégorie, et réorganisez par glisser-déposer.

Utiliser un ouvrage dans un devis

Dans l'éditeur de devis, glissez un ouvrage depuis le panneau bibliothèque vers la zone du devis. Le taux MO du devis est automatiquement appliqué.

Enregistrer un ouvrage depuis un devis

Dans l'éditeur, glissez une ligne depuis le devis vers le panneau bibliothèque. L'ouvrage est enregistré dans votre bibliothèque.

Groupes d'ouvrages (titre entier)

Vous pouvez enregistrer un titre entier d'un devis — avec tous ses ouvrages, lignes MO/matériaux et forfaits — comme groupe réutilisable (ex. « Remplacement d'un tableau électrique »).

  • Créer un groupe — glissez la ligne du titre depuis le devis vers le panneau bibliothèque. Il apparaît avec un badge groupe ;
  • Réutiliser un groupe — cliquez ou glissez l'article « groupe » : le titre complet se recrée dans le devis, avec le taux MO du devis ;
  • Modifier un groupe — sur la page Bibliothèque, un groupe s'édite comme une catégorie (lignes, prix, % matériaux) et peut être rangé dans une catégorie. Le bouton Groupe permet aussi d'en créer un de zéro.
Catégorie ou groupe ? Une catégorie sert à ranger la bibliothèque (ses ouvrages s'insèrent un par un) ; un groupe s'insère d'un bloc dans le devis.

Ajuster les tarifs

  • Taux MO — modifiez le taux en haut de la bibliothèque et appliquez-le à toutes les lignes MO en une action ;
  • % matériaux — dans chaque section, le champ % permet d'ajuster les prix matériaux de toute la catégorie.

9. Bibliothèque Aides (partagée)

Accessible depuis Aides travaux dans le menu latéral. Contrairement à la bibliothèque personnelle, elle est partagée entre tous les utilisateurs.

Elle regroupe les aides financières et subventions connues (organisme, montant, type). Les aides sont réparties en deux catégories :

CatégorieTypesZone cible dans le devis
🎁 Aides CBBFIN, FMO, FMATZone Aide travaux CBB
🏦 Aides organismesFINXZone Financements organismes

Utiliser une aide dans un devis

  1. Activez la zone de financement correspondante dans l'éditeur du devis (Aide CBB ou Financements) ;
  2. La section Aides apparaît dans le panneau bibliothèque avec les deux catégories ;
  3. Glissez une aide vers la zone correspondante, ou cliquez dessus pour l'insérer.

Une ligne liée à une aide affiche un indicateur 🔗. Le montant reste modifiable sur le devis.

Créer un utilisateur

  1. Cliquez sur Nouvel utilisateur ;
  2. Renseignez les informations (prénom, nom, email, identifiant, rôle, équipes) ;
  3. Cliquez sur Créer l'utilisateur.
Communication du compte (phase de test) — à la création, l'écran affiche une seule fois le message d'invitation complet (identifiant + mot de passe temporaire). Copiez-le et transmettez-le manuellement au nouvel utilisateur (Teams, messagerie) : l'envoi automatique par email n'est pas fiable vers les adresses internes pour l'instant. Notez le mot de passe avant de quitter la page.

Modifier ou désactiver un utilisateur

Dans la liste des utilisateurs, cliquez sur l'icône de modification pour changer le rôle, les équipes ou le responsable. Pour désactiver un compte, utilisez le bouton Désactiver — l'utilisateur ne peut plus se connecter mais ses données sont conservées.

Pas de suppression — les comptes sont désactivés, jamais supprimés, pour préserver l'historique des devis et factures.
